## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/mursal-abbas/Portfolio.git
```

Move into the project directory:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
